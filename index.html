About this Demo

This demo allows you to manipulate virtual models of various convex, regular-faced polyhedra:

  • Platonic Solids
  • Archimedean Solids
  • Prisms
  • Antiprisms
  • Johnson Solids

It was built with X3DOM, d3.js, and Bootstrap.

Geometric data taken from George W. Hart's Virtual Polyhedra.

Source on Github.
